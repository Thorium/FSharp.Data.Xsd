Function or valueDescription
( ? ) typ operation args1 args2
Signature: typ:Type -> operation:string -> args1:'T -> args2:'U -> Expr
Type parameters: 'T, 'U

Dynamic operator (?) that can be used for constructing quoted F# code without quotations (to simplify constructing F# quotations in portable libraries - where we need to pass the System.Type of various types as arguments)

There are two possible uses: typ?Name tyArgs args

tyArgs is a sequence of type arguments for method Name. Actual arguments can be either expression (Expr<'T>) or primitive values, whic are automatically wrapped using Expr.Value.

CompiledName: op_Dynamic
